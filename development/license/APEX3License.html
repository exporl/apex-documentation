ANNEX I

Description of the APEX 4 Software

APEX 4 is a software platform for auditory psychophysical experiments. It provides a generic means of setting up experiments without any programming.

The APEX 4 binaries are created by compiling the APEX 4 source code. They can be executed using a compatible operating system without any further processing.

Description of the APEX 4 User Manual

The APEX 4 User manual consists of an explanation in English of the function of APEX 4 and the structure of Experiment Files.

Description of the APEX 4 Example Experiment Files

The APEX 4 example experiment files are APEX 4 experiment files that illustrate the use of each feature of APEX 4.

Description of the APEX 4 Schema

The APEX 4 Schema describes the structure of APEX 3 Experiment Files. It specifies which XML Element can occur where and also contains documentation on each element in English. The APEX 3 Binaries use the APEX 4 Schema to test whether a given experiment file is valid.
